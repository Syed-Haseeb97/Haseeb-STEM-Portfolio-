## 💻 Running Locally

Clone the repository:

```bash
git clone https://github.com/Syed-Haseeb97/Haseeb-STEM-Portfolio-.git
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build production version:

```bash
npm run build
```
